## 主要なアルゴリズム

### 1. チーム分けアルゴリズム
```
入力: 10人のプレイヤーリスト
処理: 
  1. 1000回ランダムシャッフル
  2. 各シャッフルで青チーム5人、赤チーム5人に分割
  3. 両チームの合計MMRを計算
  4. MMR差が最小の組み合わせを保存
出力: 最もバランスの取れたチーム編成
```

### 2. MMR更新アルゴリズム（Eloレーティング）
```
入力: 勝利チーム、敗北チーム
処理:
  1. 各チームの平均MMRを計算
  2. 期待勝率を計算: 1 / (1 + 10^((敗者MMR - 勝者MMR) / 400))
  3. K-factor = 32
  4. 勝者MMR += K × (1 - 期待勝率)
  5. 敗者MMR -= K × 期待勝率
  6. MMRは0以下にならない
出力: 更新されたMMR値
```
